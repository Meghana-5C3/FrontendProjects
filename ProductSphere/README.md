# 🛒 ShopSphere – Amazon Style E-Commerce Frontend

ShopSphere is a professional Amazon-inspired e-commerce frontend web application developed using HTML, CSS, and JavaScript.

The project demonstrates real-world frontend functionalities such as product listing, category filtering, sorting, searching, shopping cart management, responsive UI design, and interactive user experience.

---

# 🚀 Features

## ✅ Product Listing
- Dynamic product cards
- Real-world product layout
- Product categories
- Product pricing & ratings

## ✅ Category Filtering
Users can filter products by:
- Electronics
- Fashion
- Gaming
- Shoes

## ✅ Product Search
- Real-time search functionality
- Search products by name

## ✅ Price Filtering
Users can filter products based on:
- ₹1000 – ₹2000
- ₹2000 – ₹3500
- ₹3500 – ₹5000

## ✅ Product Sorting
Sort products by:
- Price Low to High
- Price High to Low
- Highest Rating

## ✅ Shopping Cart
- Add to cart
- Increase quantity
- Decrease quantity
- Remove items
- Dynamic total price
- Cart item counter

## ✅ Toast Notifications
- Product added confirmation popup

## ✅ Responsive Design
- Mobile responsive
- Tablet responsive
- Desktop optimized

## ✅ Modern UI/UX
- Amazon-inspired layout
- Hover effects
- Professional design
- Interactive interface
