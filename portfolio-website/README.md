# 🌐 Personal Portfolio Website

A professional multi-page portfolio website developed using HTML, CSS, and JavaScript to showcase skills, projects, resume, and contact details with a modern responsive design.

---

# 🚀 Features

- Responsive Multi-Page Design
- About, Projects & Contact Sections
- Technical Skills Showcase
- GitHub & Live Project Links
- Resume Integration
- Social Media Links
- Smooth UI Animations
- Mobile Friendly Layout

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Font Awesome
